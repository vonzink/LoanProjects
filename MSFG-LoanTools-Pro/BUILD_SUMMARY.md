## 🚀 **How to Use**

### **Web Application**
```bash
# Start development server
npm run serve

# Open in browser
http://localhost:3000
```

### **Desktop Application**
```bash
# Start Electron app
npm start

# Build for distribution
npm run build:mac    # macOS
npm run build:win    # Windows
npm run dist         # Both platforms
```

### **Development Mode**
```bash
# Watch for changes and auto-reload
npm run watch
```
